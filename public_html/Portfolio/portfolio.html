PORTFOLIO

  • Software Engineering: Semester long collaborative team project to enhance teamwork and web application design. Learned better practices for managing object-oriented design, scalability, technical debt, patterns, etc. Positions included project manager, frontend, backend, android, and testing. The roles where I exceled were project manager, frontend, and backend. Basis of the project was to create a message board by deploying a free backend server and database on Heroku. Brains of the UI were coded in Typescript, a transpiler for more robust JS error catching. Databases held different user accounts, messages, comments, and a finite state machine implementation was used for a liking system. Accounts were manually managed through our databases and then the web application was converted to use Google OAuth 2.0 to handle login. Frontend / backend communication by transmission of Json data through HTTP routes. Our group took the project a step further by introducing Direct Messaging which had an analagous feel to GroupMe. Professor requested that the source code not be made public, so the below figures are some snapshots of the web application in action.

  • MIPS Simulator: This program is a simple simulator for a subset of MIPS instructions. The simulator is written in MIPS assembly to further gain knowledge on the stored-program concept and computer architecture. It handles R, I, and J type instructions. The example instructions loaded into memory represent a childish division program that divides 14 by 7 through repeated subtraction. Developed and tested in QtSpim. The figure below is a section of the standard output from the simulator showing the program counter, instruction fetch, register values, ALU output, etc. in hex representation for each executed instruction.
  • GitHub: mips-simulator.s
  • Microcontrollers: First program is USART implementation on Atmel AVR microcontroller using interrupts. Second program is a basic stopwatch design using USART and timer interrupts. Testing done on STK-500 evalation board with ATmega8515 microcontroller.
